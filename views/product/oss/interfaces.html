# 约束定义

# 桶名称命名规范

遵循S3标准协议:

  • 存储桶名称的长度介于 3 和 63 个字符之间，并且只能包含小写字母、数字、句点和短划线。
  • 存储桶名称中的每个标签必须以小写字母或数字开头。
  • 存储桶名称不能包含下划线、以短划线结束、包含连续句点或在句点旁边使用短划线。
  • 存储桶名称不能采用 IP 地址格式 (198.51.100.24)。

# 对象唯一关键字命名规范

遵循S3标准协议。

  • 同 桶名称 定义
  • 下列字符因为需要进行大量的特殊处理才能在所有应用程序间保持一致性，所以建议直接避免使用。需要避免的字符如下：
`	^	"	\
{	}	[	]
~	%	#	|
>	<	ASCII 
128-255 十进制

# 对象操作

# 添加对象

  • 在指定的存储桶下添加对象

    • 指定桶名称、对象唯一关键字、对象。均不允许为空
    • 不允许在一个桶下，添加两个唯一关键字相同的对象
    • 桶名称符合命名规范
    • 对象唯一关键字符合命名规范

  • 接口：com.shine.oss.OSSService#createBucket

	/**
	 * 上传数据对象
	 * 
	 * @param bucketName    目标桶的名称
	 * @param key           上传对象的object key
	 * @param filename      对象的原始文件名
	 * @param objectContent 对象的数据
	 * @return
	 */
   public RetObject<ObjectInfo> putObject(String bucketName, String key, String filename, final byte[] objectContent);

	/**
	 * 上传指定文件
	 * 
	 * @param bucketName 目标桶的名称
	 * @param key        上传对象的object key
	 * @param source     源文件
	 * @return
	 */
   public RetObject<ObjectInfo> putObject(String bucketName, String key, final File source);

	/**
	 * 从指定的输入流，上传对象。
	 * 
	 * @param bucketName 目标桶的名称
	 * @param key        上传对象的object key
	 * @param filename   源文件名
	 * @param from       源数据输入流
	 */
   public RetObject<ObjectInfo> putObject(String bucketName, String key, String filename, final InputStream from);

# 获取对象

  • 获取桶下指定的对象。
    • 根据桶名称、对象唯一关键字获取对应的对象。
    • 桶名称符合命名规范。
    • 对象唯一关键字符合命名规范。
    • 使用后需主动关闭流对象
  • 接口：com.shine.oss.OSSService#getObject
	/**
	 * 获取指定对象（文件）的输入流（读取对象信息） 在使用后，请关闭InputStream
	 * 
	 * @param bucketName 桶名称
	 * @param key 上传对象的object key
	 * @return 对象信息
	 */
   public RetObject<OSSObject> getObject(String bucketName, String key);

   /**
	 * 获取指定对象（文件）,写入到目标文件（客户端本地）
	 * 
	 * @param bucketName 桶名称
	 * @param key 上传对象的object key
	 * @param target 输出文件
	 * @return 是否获取成功
	 */
   public RetObject<Boolean> getObject(String bucketName, String key, final File target);

   /**
	 * 获取（下载）指定的对象（文件）数据，将输出流作为目标，写入数据
	 * 
	 * @param bucketName 桶名称
	 * @param key 上传对象的object key
	 * @param to 指定输出流
	 * @return 是否获取成功
	 */
   public RetObject<Boolean> getObject(String bucketName, String key, final OutputStream to);
  • 输入参数

# 检索对象

  • 检索指定桶下的对象
    • 支持桶名称及唯一关键字进行检索。返回是否存在的信息
    • 支持桶名称不为空、唯一关键字为空时，返回桶中的前200笔对象名称
    • 支持检索数据按页返回，未指定分页的情况下，最大返回200笔记录
    • 对象唯一关键字符合命名规范
    • 桶名称符合命名规范
  • 接口：com.shine.oss.OSSService#listObjects
	/**
	 * 列出桶里面的对象列表 默认返回200条记录
	 * 
	 * @param bucketName 桶名称
	 * @return 对象列表
	 */
	public RetObject<List<ObjectInfo>> listObjects(String bucketName);

	/**
	 * 列出桶里面的对象列表 根据设定的位移和记录数限制返回 最大上限不超过200条记录
	 * 
	 * @param bucketName 桶名称
	 * @param offset 偏移值
	 * @param limit 长度
	 * @return
	 */
	public RetObject<List<ObjectInfo>> listObjects(String bucketName, int offset, int limit);

	/**
	 * 根据key前缀，罗列对象 默认返回200条记录
	 * 
	 * @param bucketName 桶名称
	 * @param prefix 前缀搜索词
	 * @return
	 */
	public RetObject<List<ObjectInfo>> listObjects(String bucketName, String prefix);

	/**
	 * 根据key前缀，罗列对象 根据设定的位移和记录数限制返回 最大上限不超过200条记录
	 * 
	 * @param bucketName 桶名称
	 * @param prefix 前缀搜索词
	 * @param offset 偏移值
	 * @param limit 长度
	 * @return
	 */
	public RetObject<List<ObjectInfo>> listObjects(String bucketName, String prefix, int offset, int limit);
  • 输入参数
参数 类型 是否必须 说明
bucketName String 是 桶名称
prefix String 重载,指定方法必须 前缀搜索词
offset int 重载,指定方法必须 偏移值（从0起）
limit int 重载,指定方法必须 长度
  • 返回参数说明
List<ObjectInfo>

ObjectInfo 参数说明

参数 类型 说明

name String 对象名称
contentSize long 对象大小

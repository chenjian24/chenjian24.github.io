查询用户角色列表接口

# 查询用户角色列表接口

查询用户角色列表, 通过用户代码和角色名。

# REST API

  • 地址：/rest/user-roles

  • 请求方式：GET

  • 输入参数

参数 是否必须 说明
userCode 是 用户代码
roleName 是 角色名称，like匹配
queryStart 是 分页查询-起始记录行,默认是0
queryLimit 是 分页查询-查询记录数, 默认是-1,不限制
  • 返回参数

查询成功，返回HTTP CODE 200，返回数据JSON格式如下：

[
    {
        "roleId": -1,
        "roleName": "TestRole"
    },
    {
        "roleId": 1,
        "roleName": "leeTestRole"
    }
]

返回参数说明

参数 说明
roleId 角色ID
roleName 角色名称
  • 请求示例

curl -X GET /rest/user-roles?userCode=lee&roleName=Role

# 使用客户端调用

  • 接口：List<Role> RoleService.queryRoles(String userCode,String roleName,Integer queryStart,Integer queryLimit)

  • 输入参数：RoleQueryParam:见REST API "输入参数"说明

  • 返回参数：List<Role>

    Role:对象属性见REST API "返回参数说明"

  • 代码示例
RoleService roleService = SystemAPIFactory.getRoleService ();

List<Role> roles = roleService.queryRoles("lee", "Role",null,null);

获取用户信息接口

# 获取用户信息接口

根据用户代码获取用户信息。

# REST API

  • 地址：/rest/users/{userCode}

  • 请求方式：GET

  • 输入参数

参数 是否必须 说明
userCode 是 用户代码
  • 返回参数

查询成功，返回HTTP CODE 200，返回数据JSON格式如下：

{
	"userCode": "test",
	"userName": "测试用户",
	"mobile": "18558799999",
	"email": "425450000@qq.com",
	"deptCode": "#000000001"
}

未查询到用户信息，返回HTTP CODE 404，返回异常信息，例如：

{
    "message": "test001用户没有找到!"
}

返回参数说明

参数 说明
userCode 用户代码
userName 用户名称
deptCode 所在部门代码
mobile 手机号码
email 邮件
  • 请求示例

curl -X GET /rest/users/test

# 使用客户端调用

  • 接口：User UserService.getUser(String userCode)

  • 输入参数:见REST API "输入参数"说明

  • 返回参数：User

    User:对象属性见REST API "返回参数说明"

  • 代码示例
UserService userService = SystemAPIFactory.getUserService ();
User user = userService.getUser("test");

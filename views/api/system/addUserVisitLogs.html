新增访问日志接口

# 新增访问日志接口

新增访问日志。

# REST API

  • 地址：/rest/user-visit-logs

  • 请求方式：POST

  • 输入参数

参数 是否必须 说明
userCode 是 用户代码
clientIp 是 客户端IP
logKey 是 sessionId
logType 是 日志类型
  • 返回参数

查询成功，HTTP CODE 为200，返回数据JSON格式如下：

{
    "id": 2282,
    "userCode": "test0322",
    "clientIp": "127.0.0.1",
    "logKey": "D3d7e6a7-f544-4c72-81d6-666666666666",
    "loginDate": "2020-04-28T07:19:19.000+0000"
}

返回参数说明

参数 说明
id 表ID
userCode 用户代码
clientIp 客户端IP
logKey SESSIONID
loginDate 登入时间
  • 请求示例

curl -H "Content-Type: application/json"

-X POST

-d '{"userCode": "test0322","clientIp": "127.0.0.1","logKey": "D3d7e6a7-f544-4c72-81d6-666666666666","logType": "login"}'

"/rest/user-visit-logs"

# 使用客户端调用

  • 接口：LogService.addAuthsLog(AuthsLogAddParam authsLogAddParam)

  • 输入参数：AuthsLogAddParam

AuthsLogAddParam 对象属性见REST API "输入参数"说明

  • 返回参数：无

  • 代码示例

LogService LogService = SystemAPIFactory.getLogService();
AuthsLogAddParam dto = new AuthsLogAddParam();
dto.setUserCode("ADMIN1");
dto.setLogKey("D3d7e6a7-f544-4c72-81d6-666666666666");
dto.setClientIp("127.0.0.1");
dto.setLogType("login");
logService.addAuthsLog(dto);
